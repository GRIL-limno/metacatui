Attribute

A 'field' or 'variable' in this data file (e.g. species, site).

Attribute Name

The official name of the attribute as it appears in the data. It is usually found in the header of a data table.

Attribute Label

A short, descriptive label that is used to display the name of this attribute.

Definition

Explain the contents of the attribute fully so that a data user could interpret the attribute accurately.
